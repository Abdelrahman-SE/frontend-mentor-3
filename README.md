# Social Links Profile

A clean and responsive **Social Links Profile Card** built using **HTML5** and **CSS3** as part of a Frontend Mentor challenge.

## Overview

This project displays a personal profile card with:

- Avatar image
- Name and location
- Short bio
- A list of social media links

The main goal of this challenge is to practice **layout alignment**, **typography**, **hover states**, and **responsive spacing** using pure CSS.

## Built With

- HTML5
- CSS3
- Flexbox
- CSS Variables
- Google Fonts (Inter)
- Hover & transition effects

## Responsive Design

- The card adapts well to different screen sizes
- Uses `max-width` with flexible margins
- Centered vertically and horizontally using Flexbox
- Mobile-friendly by default

## What I Practiced

- Building a profile card UI
- Using CSS custom properties for colors
- Creating reusable button-like list items
- Smooth hover transitions
- Clean and minimal design structure

## Lessons Learned

- Better control of spacing using logical properties
- Improving hover feedback for UI elements
- Writing cleaner and more maintainable CSS
